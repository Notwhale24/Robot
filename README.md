# Robot Control System

Лабораторная работа №2: Имитация управления двигателем робота.

## Команды
- `forward <ms>` - вперед N миллисекунд
- `backward <ms>` - назад N миллисекунд
- `left <ms>` - влево N миллисекунд
- `right <ms>` - вправо N миллисекунд
- `stop` - остановка
- `status` - статус двигателя
- `help` - справка
- `exit` / `quit` - выход
